# Options

Options control theming, behavior, and timing. Set them globally via the `options` prop, or override per-step.

```tsx
<Joyride
  options={{
    primaryColor: '#e91e63',
    hideOverlay: true,
    scrollDuration: 500,
  }}
  steps={[
    {
      target: '.my-element',
      content: 'Hello!',
      // Per-step override:
      hideOverlay: false,
    },
  ]}
/>
```
