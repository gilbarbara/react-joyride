# Step

A step is a plain object that only requires two properties: <Code color="primary">target</Code> and <Code color="primary">content</Code>.

## Inherited Props

Steps inherit all [Shared Props](./props#shared-props) from the Joyride component. You can override them per-step:

- `arrowComponent`
- `beaconComponent`
- `floatingOptions` — Step-level floatingOptions are deep-merged with component-level floatingOptions.
- `loaderComponent`
- `locale`
- `styles`
- `tooltipComponent`

## Per-Step Options

Steps can override any [Options](./props/options) by setting it directly as a field on the step object.

```tsx
{
  target: '.my-selector',
  content: 'Custom behavior for this step',
  // Override options for this step only:
  skipBeacon: true,
  hideOverlay: true,
  scrollOffset: 50,
  before: async () => {
    await loadData();
  },
}
```

</Callout>
